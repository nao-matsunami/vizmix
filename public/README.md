## Quick Start / クイックスタート

### English
1. Open VizMix in Chrome browser
2. Click "Output Window" to open the output window
3. Drag & drop video files onto bank buttons
4. Use the crossfader to mix between channels
5. Click VIDEO EFFECTS to access effects panel

### 日本語
1. ChromeブラウザでVizMixを開く
2. 「Output Window」をクリックして出力ウィンドウを開く
3. 動画ファイルをバンクボタンにドラッグ&ドロップ
4. クロスフェーダーでチャンネルをミックス
5. VIDEO EFFECTSをクリックしてエフェクトパネルを開く

## Keyboard Shortcuts / キーボードショートカット

| Key | Function |
|-----|----------|
| 1-8 | Channel A banks |
| Q-I | Channel B banks |
| A/S/D | Crossfader (0%/50%/100%) |
| ←→ | Crossfader ±1% |
| Space | Tap tempo |
| F1-F6 | Effects |
| H | Help |
