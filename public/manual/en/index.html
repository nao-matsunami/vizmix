About VizMix

VizMix is a browser-based VJ (Visual Jockey) software.

  • No installation required - runs in your browser
  • 2-channel mixing - crossfade between A/B channels
  • 16 banks - 8 media slots per channel
  • BPM sync - tap tempo and auto-switching
  • MIDI support - control with hardware controllers
  • GLSL shaders - real-time graphics

Installation

Web Version (Recommended)

Simply access VizMix in your browser. No installation required.

  1. Open Chrome browser
  2. Navigate to VizMix URL (Coming soon)
  3. Start using immediately

Web version requires no additional software installation.

Download Version (Offline/Advanced Users)

To use locally, a simple web server is required.

Method 1: Python (Easy)

  1. Download and extract the ZIP file
  2. Open Command Prompt / Terminal
  3. Navigate to folder: cd VizMix-v0.8.1
  4. Start server: python -m http.server 8000
  5. Open http://localhost:8000 in browser

Method 2: Node.js

  1. Download and extract the ZIP file
  2. Open Command Prompt / Terminal
  3. Navigate to folder: cd VizMix-v0.8.1
  4. Start server: npx serve
  5. Open the displayed URL in browser

Method 3: VS Code Live Server

  1. Open the extracted folder in VS Code
  2. Install "Live Server" extension
  3. Right-click index.html → "Open with Live Server"

Note: The download version cannot be opened by double-clicking index.html. Due to browser security restrictions, access through a local server is required.

Basic Usage

Screen Layout

  • Channel A (Left) - Video source A + 8 bank buttons
  • Master (Center) - Crossfader, BPM settings
  • Channel B (Right) - Video source B + 8 bank buttons

Loading Videos

  1. Drag and drop video files onto bank buttons
  2. A thumbnail appears and the video is loaded
  3. Click bank buttons to switch videos

Supported formats: MP4 (H.264), WebM (VP8/VP9). MP4 recommended.

Note: Speed controls (reverse playback, etc.) have been hidden in v0.7.0.

Loading Shaders

  1. Drag and drop shader files (.glsl / .frag / .fs) onto bank buttons
  2. Button turns purple when shader is loaded

Crossfade

Use the center slider to mix between Channel A and Channel B.

  • Left: Channel A 100%
  • Center: 50% / 50%
  • Right: Channel B 100%

Output Window

Click "Open Output" to open the output window. Move it to the monitor connected to your projector or LED and make it fullscreen.

Media Browser

Media Browser panels are located next to Bank A / Bank B. Browse folder contents and easily assign videos to banks.

Selecting a Folder

  1. Click the "Select Folder" button
  2. Choose a folder containing video files
  3. Supported files will be displayed as thumbnails

View Modes

  • Grid View: Display thumbnails in a grid layout
  • List View: Display filenames with thumbnails vertically

Toggle between views using the icons in the panel header.

Assigning to Banks

  1. Drag a thumbnail from the Media Browser
  2. Drop it onto the desired bank button
  3. The video is loaded and assigned to the bank

Supported Formats

  • MP4 (H.264) - Recommended
  • WebM

Webcam

Use the Media / Cam toggle button below the Channel A / Channel B preview to switch to camera input.

Basic Usage

  • Media / Cam Button: Click to toggle between Media mode and Cam mode
  • Cam Mode: Display camera feed in real-time
  • Media Mode: Display normal bank media

Multi-Camera Support

  • Select which camera to use from the dropdown menu
  • Channel A and Channel B can use different cameras
  • OBS Virtual Camera and similar tools are also supported

Effects

VIDEO EFFECTS and the crossfader are applied to camera footage as well. You can apply effects to camera input just like media files.

Requirements

  • HTTPS or localhost environment required
  • Browser will request camera access permission on first use
  • If camera is not recognized, try refreshing the page

BPM Features

Setting BPM

  • Manual input: Enter value directly (40-200)
  • Tap tempo: Tap the TAP button or Space key in rhythm

Beat Indicator

Four dots light up in sync with the beat.

Auto-Switch

  1. Enable "Auto Switch" checkbox
  2. Select switch interval (1/2/4/8/16 beats)
  3. A↔B will switch automatically on the beat

Effects

VIDEO EFFECTS Panel

Control effects from the VIDEO EFFECTS panel below the bank buttons.

How to Use

  • Open/Close Panel: Click header "▼ VIDEO EFFECTS"
  • Select Effect: Click effect in list → Parameters appear
  • Toggle ON/OFF: Double-click effect, or use toggle in parameter panel
  • Adjust Parameters: Drag sliders

MIDI Support

Connection

  1. Connect your MIDI controller to PC
  2. Open VizMix (or refresh the page)
  3. Device name appears in top-right when connected

Note: MIDI only works in Chrome/Edge. HTTPS connection required.

MIDI Assignments

MIDIFunction
Note 0-7Channel A banks 1-8
Note 8-15Channel B banks 1-8
Note 16Auto-switch toggle
Note 17Tap tempo
CC 1Crossfader (0-127 → 0-100%)
CC 2BPM (0-127 → 40-200)

Testing with Virtual MIDI

If you don't have a MIDI controller, you can test using virtual MIDI software like loopMIDI (Windows).

Troubleshooting

Video not displaying

  • Check video format (MP4/H.264 recommended)
  • Try disabling Chrome hardware acceleration
  • Refresh page (Ctrl+Shift+R)

MIDI not recognized

  • Confirm using Chrome/Edge
  • Confirm HTTPS connection (or localhost)
  • Connect MIDI device before refreshing page

Performance issues

  • Lower video resolution (1080p recommended)
  • Close other tabs and applications
  • Reduce shader complexity
